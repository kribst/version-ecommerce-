Instructions

Comment ça fonctionne :

  1. Sélectionnez une ou plusieurs images depuis votre ordinateur
  2. Les images seront automatiquement associées aux catégories selon leur nom
  3. Le nom du fichier (sans extension) doit correspondre au nom de la catégorie
