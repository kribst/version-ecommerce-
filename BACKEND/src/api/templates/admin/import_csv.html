Importer des produits depuis un fichier CSV

Instructions

Le fichier CSV doit contenir les colonnes suivantes :

  • NOMS : Nom du produit (obligatoire, unique)
  • Catégories : Nom de la catégorie (sera créée automatiquement si elle n'existe pas)
  • Description courte : Description courte du produit
  • Description : Description complète du produit
  • PRIX : Prix du produit (nombre)
  • IMAGES : URL de l'image à télécharger
