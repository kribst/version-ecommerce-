Instructions

Le fichier CSV doit contenir les colonnes suivantes :

  • nom : Nom de la catégorie (obligatoire, unique)

  • sous parent : Nom d'un niveau intermédiaire (optionnel, enfant du parent)
  • image : URL de l'image de la catégorie (optionnelle)
